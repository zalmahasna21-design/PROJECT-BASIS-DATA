# PROJECT-BASIS-DATA
# Deskripsi Project
Sistem Informasi Logistik & Pengiriman "LogiTrack" adalah sistem basis data yang dirancang untuk mengelola proses pengiriman paket secara terstruktur dan efisien. Sistem ini menangani seluruh alur data mulai dari pendaftaran pengirim dan penerima, pembuatan paket dengan nomor resi unik, penjadwalan kurir, perhitungan tarif berdasarkan rute dan berat, hingga pelacakan status paket secara realtime.
